Leenfiets website
Leenfiets is a fake company that me and some schoolmates made up. Leenfiets is Dutch for borrow bike and as the name suggests you can borrow bikes from private individuals through the website that we made.

The main focus of this project was to learn the basics of php, databases and learn how to work in a team using scrum.

Even though it is not the prettiest website I am very happy to show the result because the website contains all the functionality for the whole borrowing process to work and more.
Date: 2019

A deposit is used to ensure that the provider gets the bike back in good condition. The lessor receives a fee for lending his bicycle.

What I mainly worked on is reserving and borrowing the bicycles. A QR code is used to borrow a bicycle. You check in by scanning the lessors QR code and you scan the lessors QR code again when you have returned the bike. This way we can easily keep track of whether the bicycle is already being borrowed and we can ensure that the provider does not report the bicycle as stolen and the borrower is forced to pay when he returns the bike.

In addition, I have also made a calendar in javascript in which you can see whether the bike you want to borrow is available on that date.

I also added a system so that you can put money in your account and borrow bicycles with that money. This is fictitious money.


I also contributed to the other functionalities by thinking along with my teammates for ways of implementing code and solving problems.

Other functions of the websites that I have not mentioned yet are:
  • Users can register
  • Admin dashboard for adding more options for he bikes and see users
  • Review page
  • News page
  • Search for bikes with a filter
  • Email verification
